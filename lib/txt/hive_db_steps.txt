Step 2 : Create model. (Here person_model.dart)
# Model is necessary for organizing data in hive db.
See person_model.dart (Change according to requirements)



Step 3 : Open Terminal and run below command 
command => flutter packages pub run build_runner build

'modelname.g.dart' file will be created in the same directory, where the model file is.
Here it is person_model.g.dart



Step 4 : In main.dart file write below code

void main() async {
  await Hive.initFlutter();
  Hive.registerAdapter(PersonAdapter()); // PersonAdapter from modelname.g.dart file
  await Hive.openBox<Person>('persons'); // Person from model file
  runApp(MyApp());
}
